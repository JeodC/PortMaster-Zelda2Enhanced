## Installation
Download the .zip from https://hoverbat.itch.io/ziiaol and put in the port folder.

## Controls
| Button | Action |
|--|--|
|START|Menu|
|SELECT|Cast Spell|
|D-PAD|Move|
|A|Jump|
|X|Attack|
|L2|Options Menu|
